# Task 2 - Responsive Contact Form and To-Do List

## Project Overview
This project was developed as part of Task 2 of my Web Development Internship at ApexPlanet Software. It demonstrates the use of HTML, CSS, and JavaScript to create a responsive web application with form validation and dynamic DOM manipulation.

## Features
- Responsive Navigation Bar using Flexbox
- Contact Form with:
  - Name
  - Email
  - Phone Number
  - Message
- JavaScript Form Validation
  - Required field validation
  - Email format validation
  - Success message
- Responsive Layout using CSS Grid
- Media Queries for mobile responsiveness
- Dynamic To-Do List
  - Add Tasks
  - Delete Tasks

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- DOM Manipulation

## How to Run
1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server or any web browser.
4. Test the contact form and to-do list features.
